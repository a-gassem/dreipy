About DRE-ipy

DRE-ipy is a free and open-source implementation of the Direct Recording Electronic with Integrity and Enforced Privacy (DRE-ip) protocol written using Python 3 and Flask. It has added various features on top of previous implementations of DRE-ip in order to make it more usable, including:

  1. Allowing for multiple-choice questions.
  2. Letting elections have more than one question.
  3. Providing basic data visualisation for the results after the election.

For questions about how DRE-ip works and the security of DRE-ip, please look at the Frequently Asked Question section below.

Frequently Asked Questions:

  2. How does an election work?
  3. How can I know that my vote was counted?
  4. What do I do if my vote was recorded incorrectly?
